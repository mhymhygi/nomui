# API

| 参数                | 说明                 | 类型        | 默认值   |
| ------------------- | -------------------- | ----------- | -------- | --- | --- | ---- | ------- |
| size                | 设置头像组的大小     | `number     | xs       | sm  | lg  | xl ` | default |
| maxCount            | 显示的最大头像个数   | `number `   | -        |
| maxPopoverPlacement | 多余头像气泡弹出位置 | `top        | bottom ` | top |
| items               | 显示的子头像列表     | `Avatar[] ` | []       |
