字段组，组合一组字段，通过配置 `fields` 来配置一组子字段。
字段组继承自 `Field`。

## API

| 参数            | 说明                   | 类型            | 默认值      |
| --------------- | ---------------------- | --------------- | ----------- |
| fields         | 子字段数组    | `Array`         | -           |
| inline         | 内联布局    | `Boolean`         | -           |
| striped         | 条纹样式，标签栏有背景色    | `Boolean`         | -           |
| line         | 线条样式    | `splitline`\|`outline`         | -           |

## Method

| 方法名称 | 说明                    | 参数          |
| -------- | ----------------------- | ------------- |
| getField | 获取子字段，参数为字段名，可有圆点分隔，例如`foo.bar`来获取多层级的子孙字段 |               |
