导航条
导航条一般用在网站的主导航上，一般包括标题，导航内容和工具栏三部分

## API

| 参数       | 说明           | 类型     | 默认值 |
| ---------- | -------------- | -------- | ------ |
| caption      | 标题   | `Caption` | -      |
| nav   | 导航内容   | `Array` \| `Props` | -      |
| tools   | 工具栏   | `Array` \| `Props` | -      |
| captionBefore       | 标题前内容 | `Array` \| `Props` | -      |
| captionAfter      | 标题后内容 | `Array` \| `Props` | -      |
