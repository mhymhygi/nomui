## API

### Modal

| 参数      | 说明                      | 类型      | 默认值          |
| --------- | ------------------------- | --------- | --------------- |
| container | 指定 loading 效果附着元素 | `object`  | `document.body` |
| backdrop  | 是否带遮罩层              | `boolean` | `false`         |
